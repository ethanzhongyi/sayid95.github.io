hexo 踩过的一些坑

做技术还是要写点东西的，俗话说得好好记性不如烂笔头，

以前都是写在印象笔记上的，后来发现一个问题，笔记上的东西太多，

写的时候不会那么认真，毕竟只是自己看到，二来查找不太好用。

blogs 上还是会仔细写的，因为看的人不止是自己了，再者可以给需要的人一点思路。

hexo博客的搭建等问题可以 看官网，我就不去赘述了

  • 一 部署问题

hexo博客系统是挂载在GitHub上的，so需要在配置里面去部署

  • 二 配置问题

搭建好了以后我们也会不断去_config.yml 文件里配置一些东西，当然不小心就会出错的，error当然也不好找，用这个把_config.yml 配置复制到这里去检测配置上的错误

  • 三 报错 Cannot find module ‘hexo-util’

我们可以输入命令 npm install – save-dev hexo-util

重新安装模块后，hexo clean hexo d 等操作都不会报错了！

  • 四 迁移hexo

    换电脑后还需要重新设置pub SSH KEY到GitHub，这里是具体做法
